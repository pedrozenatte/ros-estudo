# As distribuições do ROS

## O que são as distribuições do ROS?
São conjuntos de pacotes que são entregues a comunidade a cada período de tempo. 
Um exemplo, foi a distribuição "ROS Box Turtle", disponibilizada em 2010. 

**ATENÇÃO:** A última distribuição do ROS1 (Noetic Ninjemys) foi atualizada até maio de 2025 para o Ubuntu Focal Fossa (20.04 LTS). A partir desse período, tudo é feito para o ROS2.
